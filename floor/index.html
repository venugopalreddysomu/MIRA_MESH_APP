Calibration Mode

Calibration mode pauses all mesh activity so SPP communication is not interrupted during configuration. Writes to Meter Config and the Calibration Table are only allowed while calibration mode is active.

Enter the secret code to unlock. The mode is cleared on SPP disconnect or reboot — it is not persisted to flash.

✓ Calibration mode is ACTIVE — mesh TX/RX is paused, writes are enabled.
⚠ Writes require Calibration Mode to be active.
